# ===========================================================================
# 1. ГЕОМЕТРИЯ ДЛЯ КЕРАМИКИ (Физика процесса)
# ===========================================================================

- Устойчивость: Foot ring, внутри вогнутое дно.
- Свод дна: В 2D-профиле центр дна: [0, foot_ring_height + wall].
- Антизакусывание: draft angles или chamfer/rounding.
- $slop = 0.01; — микро-зазор для CSG.
- Безопасность скруглений: max(0, calculation) для радиусов.
- Галтели: отрицательный rounding в месте контакта.

# ===========================================================================
# 2. ПРИНЦИП ЭКВИДИСТАНТЫ (Шликер)
# ===========================================================================

- Внутренняя полость = Внешняя форма минус толщина стенки (wall).
- Логика rounding1: rounding_outer - wall.
- max(0, ...) обязателен для вычисляемых радиусов.
- Приоритет rotate_sweep() по 2D-профилю (исключает "толстое дно").

# ===========================================================================
# 3. СПЕЦИФИЧНЫЕ ПРИЁМЫ КЕРАМИКИ
# ===========================================================================

- Foot ring: выступающее кольцо в основании, высота 2-4 мм, ширина 3-5 мм.
- Вогнутое дно: углубление 1-2 мм внутрь от foot ring для устойчивости при обжиге.
- Равномерная толщина стенки: wall = 3-5 мм для шликера.
- Углы наклона (draft): минимум 2-3° для извлечения из формы.
- Скругления в местах соединения стенки и дна: R >= wall/2.
